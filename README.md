# Abdul Musa

High-end portfolio website for product developer Abdul Musa.

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Lenis (smooth scroll)

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
npm start
```

## Design Principles

- Dark, minimal, editorial aesthetic
- Calm, confident, restrained interactions
- High-quality motion and transitions
- Large spacing, clean typography
- Projects presented as artifacts, not cards
- No marketing language or persuasion
